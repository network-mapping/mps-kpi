MPS Report Generator EXE Documentation

Using the windows mps_reporter_cli.exe


Step 1. Hit download to get the zip file


This zip contains everything you need to use the exe.
If you have already got the zip file downloaded and extracted on your PC you can skip this step.


Step 2. Right click the zip and extract it


The correct extract option should result in a folder of the same name as the zip


Step 3. Place Quickbooks reports into the inputs folder within the extracted folder


n.b Got the files somewhere else? Use shift + right click on the folder containing your Quickbooks reports and select 'copy as path' and use this in place of ./input in the command in Step 4.


Step 4. Run the CLI command


./mps_reporter_cli.exe --config config.yaml --input_path ./inputs --output_path ./outputs


Step 5. Check the output folder for your result


there should now be a file in the format mps_report.yyyy.mm.dd.csv
